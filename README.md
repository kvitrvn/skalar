# Skalar

A simple project manager

![CI](https://github.com/kvitrvn/skalar/workflows/CI/badge.svg)

## Getting Started - DEV environment

```bash
# build containers
make build
# start environment
make up
# show logs
make logs
# stop environment
make down

# show all available commands
make
```
